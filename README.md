# SWG, a new C++ based shell

## Purpose
- This is a semi-passion project so that I can explore the world of Linux & systems programming
- Everything is written in C++, and will be built with Makefile, using gcc(g++) as a compiler
- Currently, everything is tested on MacOS, but I would love to branch out & support Linux (based on my raspberry pi)

## Currently Implemented
- I have a sample string parser to parse tokens into executing files/programs
- I have a locally stored deque to hold all the previous commands (max hist. of 20);

## Features I Want to Implement
- I want to create a customizable terminal window, that will allow for color customization
- I also want to potentially support tmux natively with this shell, to make it an easier and seamless developer experience
  - This is mainly because I keep forgetting to run tmux when I open up terminal LOL
